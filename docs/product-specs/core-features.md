# Core Features — MVP

## Feature 1: One-Click Content Import

**As a** content creator
**I want to** paste a URL or text
**So that** the AI can understand my source content

**Acceptance Criteria:**
- [ ] Input box accepts URLs (blog posts, articles, newsletters)
- [ ] Input box accepts raw text / markdown
- [ ] AI extracts: title, author, main content, key points
- [ ] Shows preview of extracted content before transformation

## Feature 2: AI Content Transformation

**As a** content creator
**I want to** generate platform-optimized versions of my content
**So that** I can publish across multiple channels without rewriting

**Platform Outputs (MVP):**
- **Twitter/X Thread** — 5-15 tweets, numbered, with hook and CTA
- **LinkedIn Post** — Professional long-form post with formatting
- **Newsletter Summary** — Email-friendly digest version
- **Instagram Caption** — Short, engaging, hashtag-optimized
- **SEO Meta** — Title, description, keywords for blog

**Acceptance Criteria:**
- [ ] User selects which platforms to generate for
- [ ] Each output is in the correct format for that platform
- [ ] AI maintains consistent voice/tone across all outputs
- [ ] Tone adjustable: Professional, Casual, Humorous, Authoritative

## Feature 3: Preview & Edit

**As a** content creator
**I want to** preview and edit generated content before exporting
**So that** I can tweak the AI output to match my voice

**Acceptance Criteria:**
- [ ] Live preview of each generated piece
- [ ] Inline editing capability
- [ ] Regenerate button per platform
- [ ] Copy to clipboard / download

## Feature 4: History & Library

**As a** content creator
**I want to** access my previously generated content
**So that** I can reuse or republish

**Acceptance Criteria:**
- [ ] History of all generated content
- [ ] Search/filter by date, platform, topic
- [ ] Re-generate from existing source

## Feature 5: User Accounts & Subscription

**As a** user
**I want to** create an account and subscribe
**So that** I can use the service beyond the free tier

**Tiers:**
| Tier | Price | Credits/mo | Features |
|------|-------|------------|----------|

| Pro | $9 | 30 | All platforms, no watermark, history |
